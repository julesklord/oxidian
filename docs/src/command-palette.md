# Command Palette

The Command Palette centralizes access to Zed actions. Toggle the palette using {#kb command_palette::Toggle}.

![The opened Command Palette](https://zed.dev/img/features/command-palette.jpg)

Type `new file` to filter the list to {#action workspace::NewFile}. Pressing Return creates a buffer.

Execute instructions formatted as `zed: ...` or `editor: ...` through this interface.
