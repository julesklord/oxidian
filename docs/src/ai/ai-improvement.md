# Zed AI Features and Privacy

## Overview

Zed includes these AI features:

- [Agent Panel](./agent-panel.md)
- [Edit Predictions](./edit-prediction.md)
- [Inline Assist](./inline-assistant.md)
- Auto Git Commit Message Generation

Zed does not store prompts or code context. AI providers like Anthropic, OpenAI, Google, or xAI receive data to generate responses and discard it after use. Zed uses data for feature improvement only if you share it through [ratings](#ai-feedback-with-ratings) or [edit prediction training](#edit-predictions).

Zed supports multiple models. You may use API keys or Zed hosted models without data retention.

### Data Retention and Training

The Agent Panel supports these options:

- [Zed hosted models](./subscription.md)
- [External AI services via API key](./llm-providers.md)
- [External agents](./external-agents.md) via ACP

Hosted model providers must guarantee they will not train models on user content.
