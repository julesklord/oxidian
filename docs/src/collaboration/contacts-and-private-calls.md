# Contacts and Private Calls {#contacts}

Private calls allow collaboration outside of channels. Add people to your contact list to start sessions.

## Adding a Contact

1. Open the Collaboration Panel.
1. Click the `+` icon next to `Contacts`.
1. Search for a GitHub handle. Contacts must have a Zed account and an active GitHub session.
1. Zed sends a notification.
1. Both users join each other's lists once the recipient accepts.

## Private Calls

1. Locate an online contact in the Collaboration Panel.
1. Click the `...` menu next to their name.
1. Select `Call <username>`.
1. Add more participants by clicking names in the Collaboration Panel during an active call.

Private calls support the same collaboration tools as [channels](./channels.md). These sessions do not include channel notes.
