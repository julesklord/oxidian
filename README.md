# 💎 Oxidian

**Native Markdown knowledge base built on the Zed editor.**

[![Oxidian](https://img.shields.io/badge/Powered%20by-Zed-blue)](https://zed.dev)

## Vision
Oxidian provides a high-performance knowledge management system without Electron bloat. It uses the **GPUI** rendering engine and the **Rust**-native architecture of Zed for fluid note-taking.

## Core Principles
- **Native Performance:** Replaces browser-based overhead with native execution.
- **Agent Integration:** Optimized for orchestration via AI agents and MCP.
- **Data Ownership:** Uses standard Markdown files stored locally.
- **Zed Architecture:** Extends Zed without forking the source.
